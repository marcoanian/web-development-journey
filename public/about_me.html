About Me

Hello, my name is Marco.

I am a software development student with a strong interest in programming, web development, artificial intelligence, and local software applications.

I am currently studying software development at university while also completing practical courses in Python and full-stack web development. My goal is not only to understand programming theory, but also to apply what I learn by building real projects.

I began my development journey with Python, where I worked on desktop applications, automation tools, graphical user interfaces, computer vision projects, and local AI applications. I am now expanding my skills into web development by learning HTML, CSS, JavaScript, backend development, and databases.

I created this website as a learning project and as a record of my progress. Each version of the website represents a different stage in the history of web development and a new stage in my own development as a programmer.

I believe that making mistakes, testing ideas, and improving existing code are important parts of learning. For this reason, the projects on this website will show not only finished results, but also how my skills and understanding develop over time.

My long-term goal is to create useful software and web applications for real-world problems. I am especially interested in building practical tools for individuals, small businesses, and local organizations.

This website will continue to grow as I learn new technologies, complete new projects, and gain more experience as a software developer.
